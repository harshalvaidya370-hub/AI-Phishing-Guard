# 🛡️ AI Phishing Guard

A simple Machine Learning-based web application that detects whether a given text message is **phishing** or **safe**.

This project uses Natural Language Processing (NLP) techniques with a basic ML model integrated into a Flask web app.

---

## 🚀 Features

* 🔍 Classifies text as phishing or safe
* 🧠 Uses NLP (CountVectorizer) for text processing
* 🌐 Web interface built with Flask
* ⚡ Instant prediction on user input
* 💡 Lightweight and easy to run

---

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* HTML

## ⚙️ How to Run Locally

1. Clone the repository:

   ```
   git clone https://github.com/harshalvaidya370-hub/AI-Phishing-Guard.git
   cd AI-Phishing-Guard
   ```

2. Install dependencies:

   ```
   pip install scikit-learn flask
   ```

3. Run the app:

   ```
   python app.py
   ```

4. Open in browser:

   ```
   http://127.0.0.1:5000
   ```

---

## 📸 Output

* Enter any message in the input box
* The app will predict whether it is **Phishing** or **Safe**

---

## 📌 Limitations

* Model is basic and trained on limited data
* Accuracy can be improved
* No model saving (re-trains on every run)
